## Color Usage

### Semantic Colors

```
Success (Green #10b981):
  - Passed checks
  - Completed steps
  - Success messages
  - Launch button

Warning (Orange #f59e0b):
  - Pending checks
  - Important notices
  - Non-critical alerts

Error (Red #ef4444):
  - Failed checks
  - Error messages
  - Required indicators
  - Critical alerts

Primary (Blue #3b82f6):
  - Action buttons
  - Links
  - Focus states
  - Brand elements

Background/Surface:
  - Dark navy (#0f172a) - App background
  - Slate (#1e293b) - Cards, surfaces
  - Darker slate (#334155) - Hover states
  - Border gray (#475569) - Borders
```

### Text Hierarchy

```
H1 (2rem):     Screen titles
H2 (1.5rem):   Section headers
H3 (1.25rem):  Card titles
Body (1rem):   Primary content
Small (0.875rem): Secondary info
Code (Courier):   Commands, paths
```
